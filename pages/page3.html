Share Flex Message(แชร์ Flex)

ระบบ สร้างและแชร์ Flex Message ไปยังห้องแชท LINE
พัฒนาโค้ดโดยใช้ ChatGPT เนื่องจากโค้ดจากผู้เชี่ยวชาญมักไม่แจกฟรีขี้งกเกิน

ทดลองใช้งาน LIFF
กดที่รูปภาพเพื่อ Copy ลิงก์ LIFF

*หลังจากนั้นนำไปวางในช่องแชทไลน์ของเราและทดสอบส่ง Flex ได้เลย

- ตัวอย่างการสร้าง Github และ LIFF หาดูช่อง Youtube นะครับ

1. การสร้าง LIFF

กดที่ลิงก์ Github ด้านล่างเพื่อเอาโค้ด และสร้าง Github ของตัวเองเพื่อนำ LIFF ไปใช้
- ตัวอย่างการสร้าง Github หาดูช่อง Youtube นะครับ

Github

2. เปลี่ยน LIFF ID

หลังจากได้โค้ดก่อนหน้าที่เอามาจาก Github แล้วให้ทำการเปลี่ยน
LIFF ID ของ LIFF ของบอทที่สร้างไว้
- การสร้าง LIFF หาดูจาก Youtube นะครับ

LINE Developers

3. นำไปใส่ LINE Developers

จากขั้นตอนที่ 2 เมื่อเปลี่ยน LIFF ID เสร็จแล้ว ก็นำลิงก์ Github
มาใส่ใน Liff ของ LINE Developers

LINE Developers

4. นำลิงก์ LIFF ไปใช้งาน

หลังจากนำลิงก์ Github มาใส่ Endpoint URL เรียบร้อยแล้ว
ก็นำลิงก์ LIFF URL ของเรามาใช้ได้เลย
